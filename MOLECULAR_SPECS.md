### 按鈕

#### Primary 主要按鈕

**Large / Desktop（主要行動場景）**

- **邊框：** `box-shadow: 0 0 0 1.5px rgba(232, 112, 42, 0.20)` (隱性結構邊界)
- **陰影：** `box-shadow: 0 0 12px rgba(232, 112, 42, 0.30)` (待機脈動基底)
- **圓角：** 左上 & 右下 2px / 右上 & 左下 8px (非對稱切角)
- **邊距：** `padding: 14px 32px`；`min-width: 200px`；`height: 52px`
- **背景色：** **Accent-01 (#E8702A)**
- **物質狀態系統：** 疊加 [Mat-01] -5° 各向異性拉絲底質。表面透過 [Mat-05] 貼圖層執行 `mask-image: radial-gradient`，在四角區塊將遮罩權重提升 20% 以實施顆粒增壓。
- **動能邏輯：**
    - **待機：** [An-02] 執行互質干涉頻閃 (flash-A/B/C)。Peak 瞬間透過 **專屬偽元素 `opacity` 變化與 `mix-blend-mode: screen`** 產生白熱化躍升感，取代直接調用濾鏡函式。維持 ±0.2px 隨機位移與 [Mat-03] 氧化膜的 `hue-rotate(±3deg)` 緩動。
    - **Hover：** 啟動 [Int-01] 熔接色散穿透。核心 [Opt-02] 於 40ms 內執行透明度從 0 至 1 的閃現，色溫感由 2000K 轉向 8000K。邊緣 [Opt-01] 擴張 3px 並疊加 [Mat-03] 氧化膜的高感激發。
    - **Active：** 執行 [Int-02] 液態鋼片彈震。產生 `scale(0.96, 0.98)` 非對稱變形與 [Mat-02] 0.8% 物理位移，並在點擊中心釋放 [Opt-04] 重力折射漣漪（以透明度與縮放實作，嚴禁觸發重型模糊重繪）。
    - **Focus：** [Int-03] 橙色 outline 即時切入。配合 [An-04] 幾何折線從左上角以 250ms 剛性步進咬合，轉折點觸發 40ms 減速幀（CSS 階梯函數實作）。
    - **Click：** 觸發 [Mo-01] 鋼刃切入動勢。執行 120ms 的偽元素白光閃擊，隨後進行向心空間坍縮並落定於 [Mat-01] Opacity 3% 的穩態。
- **圖示與文字：**
    - **佈局結構：** 文字居中；Icon 採水平線性排列。
    - **顏色：** **Bg-01 (#0A0C0F)** 
    - **字型、字體、文字尺寸：** Noto Sans TC；400 (Regular)；14px
    - **間距：** Icon 尺寸 16px；間距 8px
